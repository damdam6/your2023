## convention

[AngularJS Git Commit Message Conventions](https://gist.github.com/stephenparish/9941e89d80e2bc58a153)을 참고함

```
<type>(<scope>): <subject> (<issue>)

<body>
```

### type

- feat (feature)
- fix (bug fix)
- docs (documentation)
- style (formatting, missing semi colons, …)
- refactor
- test (when adding missing tests)
- chore (maintain)

### scope

- 파일명 기본
- 폴더 전체 파일 수정시 - 폴더명
